# 📈 TradeIQ – AI-Powered Stock Market Predictor  

**TradeIQ** is a machine learning-powered stock market prediction platform designed to analyze financial data and predict stock prices with high accuracy. Built with **Python, Scikit-learn, TensorFlow, and Streamlit**, it enables users to explore, visualize, and forecast stock trends in real time.  

## ✨ Features  

- 🤖 **AI/ML Model**: Time-series forecasting with 85%+ accuracy on historical S&P500 datasets.  
- 🔮 **Stock Prediction**: Predict over 100 global stocks using ticker input.  
- 📊 **Interactive Charts**: Dynamic visualizations powered by Matplotlib & Pandas.  
- ⚡ **Real-Time Data**: Integrated yFinance API for live market updates.  
- 🔍 **Efficient Training**: Feature engineering reduced training time by 25%.  
- 📈 **Stable Forecasting**: Improved prediction stability by 18%.  
- 🖥 **User-Friendly App**: Seamless Streamlit UI for quick exploration & predictions.  

## 📙 Description  

**TradeIQ – AI-Powered Stock Market Predictor** (January 2025 – May 2025)  

_Python, Scikit-learn, TensorFlow, Streamlit, Matplotlib, yFinance_  

– Built an AI/ML model for stock price prediction using time-series forecasting, achieving accuracy above 85% on historical S&P500 datasets.  
– Engineered an interactive web application with Streamlit, enabling users to seamlessly analyze and accurately predict over 100 global stocks via ticker input.  
– Integrated yFinance API to fetch real-time data, visualized through dynamic charts (Matplotlib, Pandas, NumPy) for market trend analysis.  
– Enhanced model performance with feature engineering, reducing training time by 25% and improving prediction stability by 18%.

## ⚙️ Installation & Setup  

1️⃣ **Clone the Repository**  
```bash
git clone https://github.com/your-username/TradeIQ.git
cd TradeIQ
````

2️⃣ **Install Dependencies**

```bash
pip install -r requirements.txt
```

3️⃣ **Run the App**

```bash
streamlit run app.py
```
## 🚀 Usage

* Launch the app → Enter any **stock ticker** (e.g., `AAPL`, `TSLA`, `GOOGL`).
* Get **real-time financial data** with yFinance.
* View **dynamic charts** showing stock history & trends.
* See **predicted future stock prices** generated by the ML model.

* **85%+ accuracy** on historical S\&P500 datasets.
